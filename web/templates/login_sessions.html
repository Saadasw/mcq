About Login Sessions

  • Active Sessions: Currently logged-in users
  • Terminated Sessions: Logged out or force-logged-out users
  • Last Activity: Updated on every page request
  • Force Logout: Immediately terminates a user's session
  • Auto Cleanup: Sessions inactive for 24+ hours are automatically removed
  • Concurrent Sessions: Old sessions are terminated when user logs in again

Note: Session data stored in web/login_sessions/active_sessions.csv
